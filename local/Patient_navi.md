# Patient Navigation Platform (Patient_navi)

## Version 1 (MVP)
- Patient registration and case management
- Medical document upload
- AI-assisted report summarization (Ollama)
- Hospital directory
- Financial assistance directory
- Follow-up tracking
- FastAPI + SQLite + HTMX + Bootstrap

## Version 2
- RAG-powered document search
- English/Tamil explanations
- Hospital recommendation engine
- Funding recommendation engine
- PDF case summaries
- ChromaDB integration
- Knowledge library

## Version 3
### Vision
AI-assisted patient care coordination ecosystem.

### Modules
1. Clinician Review Portal
2. Volunteer Management
3. NGO & Funding Partner Portal
4. Treatment Journey Timeline
5. AI Case Navigator
6. Patient Education Center
7. Document Intelligence
8. Smart Funding Advisor
9. Hospital Intelligence Module
10. Case Collaboration

- HTMX + Bootstrap

### Roles

### Key Outcomes
